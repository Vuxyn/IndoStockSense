# Notebooks

The main course demo notebook should be created here as:

```text
pempar_demo_full.ipynb
```

Notebook design:

1. Project overview
2. Environment setup
3. Dataset loading
4. Exploratory data analysis
5. Text preprocessing
6. Genetic Algorithm design
7. Sequential GA benchmark
8. Parallel GA benchmark with PySpark
9. IndoBERT-LoRA fine-tuning with CUDA
10. Model evaluation
11. Sentiment inference demo
12. Future deployment plan

The notebook should contain full code for presentation, but functions should be written modularly so they can later be moved into `src/indostocksense`.
